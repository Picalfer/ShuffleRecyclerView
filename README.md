# ShuffleRecyclerView

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Тестовое приложение для изучения работы DiffUtil с RecyclerView и анимацией перемещения элементов.

## О проекте

Приложение демонстрирует эффективное обновление RecyclerView с помощью DiffUtil. При нажатии кнопки элементы перемешиваются с плавными анимациями.

**Основной функционал:**
- RecyclerView с 30 элементами
- Кнопка для перемешивания элементов
- Использование DiffUtil для эффективного обновления
- Плавные анимации перемещения элементов

## Технологии

* Kotlin
* RecyclerView
* DiffUtil
* ViewHolder
* List animations
